End-of-Turn Detection: Solution Summary

Problem framing

The task is to decide whether a pause in speech is a true end of turn or a hold pause. A silence-only heuristic is too weak because long hold pauses and true end-of-turn pauses can both be long and quiet. The assignment therefore needs a signal from the speech before the pause.

What was done by the human

  • Provided the assignment framing and the starter pipeline.
  • Defined the scoring objective: minimize mean response delay while keeping false cutoffs under budget.
  • Specified the deliverables: baseline, improved model, predictions for both languages, summary, run log, and notes.

What was done by the coding agent

  • Implemented a reusable feature extractor in starter/features.py that derives prosodic summaries from the last ~1.5 s of speech before each pause.
  • Updated the training script in starter/train.py to train a small, interpretable classifier and save a reusable model.
  • Created starter/predict.py so the model can run on a new folder with the same labels schema via python predict.py --data_dir <folder> --out predictions.csv.
  • Generated predictions for both provided language folders: eot_data/english/predictions.csv and eot_data/hindi/predictions.csv.
  • Produced the required documentation files: RUNLOG.md, NOTES.md, and this summary page.

Model signal

The final model uses prosodic cues from the speech immediately preceding the pause:

  • Multi-window energy trends from the last 100 ms, 250 ms, and 500 ms before the pause
  • Voiced/unvoiced ratio and voiced-region duration
  • Pitch behavior near the pause, including final pitch and pitch slope
  • Speaking-rate-style voiced-frame density and pause context

These features are more informative than silence alone because true EOTs often show a more terminal prosodic pattern, whereas hold pauses often have more hesitant or less terminal cues. The design is also explicitly causal and therefore suitable for a live assistant.

Why this beats the status quo

The baseline only sees silence and therefore cannot distinguish a long hold pause from a true end-of-turn. The new approach uses the acoustic structure of the preceding speech, which makes the decision more aligned with human turn-taking behavior. That yields a more sensible and robust agent policy while keeping the model simple and conservative to avoid false cutoffs.
